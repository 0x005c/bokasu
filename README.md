# bokasu
A simple blur/mosaic filter

## How to install
### Linux
Run commands below.
Note that you can replace `~/.local/bin/` with any directory name included in `$PATH`.
```
git clone https://github.com/0x005c/bokasu.git
cd bokasu
dub build
cp bokasu ~/.local/bin/
```

## How to use
See `bokasu -h`.
